# このソフトウェアについて

d3.jsで作成したテーブルのソートキー項目に▲▼マークをつける。

# 開発環境

* Linux Mint 17.3 MATE
* Firefox 42.0
* [d3.js](https://d3js.org/) [v4.5.0](https://cdnjs.cloudflare.com/ajax/libs/d3/4.5.0/d3.min.js)
